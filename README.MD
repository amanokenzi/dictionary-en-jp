# INSTALL

```sh:shell
$ git clone https://github.com/amanokenzi/dictionary-en-jp.git
$ cd dictionary-en-jp
$ alias dict="python dictionary.py"
```
# USE

```sh:shell
$ python3 dictionary.py [word you want to search]
<meaning>
```

> EXAMPLE
> ```sh:shell
> $ python3 dictionary.py python
> 【動物】大蛇,ニシキヘビ
> ```
